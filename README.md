# 天氣系統 B6.8.1

## 上傳 Git
將此資料夾內所有檔案放到 Git 專案根目錄。

## Firebase
1. 開啟 `firebase-config.js`
2. 貼上 Firebase config
3. 將 `firebaseEnabled` 改成 `true`
4. 設定 Firestore Rules 與 Auth 權限
